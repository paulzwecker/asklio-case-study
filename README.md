# asklio-procurement

Case Study: Beschaffungstool mit Next.js (Frontend) und FastAPI (Backend).

## Struktur

- `frontend/` – Next.js App (Request-Formular, Übersicht, UI)
- `backend/` – FastAPI API (Request-Handling, PDF-Parsing, Commodity Logic)
- `docs/` – Case-Study-Unterlagen (z.B. Angebots-PDFs)
